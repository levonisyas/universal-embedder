# universal-embedder
Home Assistant card for embedding cards directly

~~~yaml
card_mod:
  class: test-red
  style: |
    .test-red {
      position: absolute !important;
      top: 30% !important;
      right: 5% !important;
      width: 400px !important;
      height: 300px !important;
      z-index: 1 !important;
    }
~~~
